CSS Components

Panels

Wells

Buttons

Button classes can be used on <a>, <button> or <input> element.

Tables

Bootstrap features several types of tables to display data on screen.

Forms

Make sure you review the documentation for all types of Bootstrap forms.

Containers

Bootstrap requires a containing element to wrap site contents and house their grid system.

Grids

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases.
